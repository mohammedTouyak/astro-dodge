# ASTRO DODGE 🚀

ASTRO DODGE is a 2D arcade survival game built with **Unity and C#**.

The player controls a spaceship that must dodge incoming asteroids while achieving the highest score possible.

---

## 🎮 Gameplay

- Control a spaceship
- Dodge falling asteroids
- Survive as long as possible
- Compete on the leaderboard

---

## 🧠 Features

- Player movement system
- Asteroid spawning system
- Score system
- Leaderboard
- Difficulty scaling
- Game over system

---
